How to use this dashboard

Loading data

Click Load data file to browse to your Excel workbook (.xlsx or .xlsm). The workbook must contain sheets named exactly: Fixed Asset Register (IT), Software Asset Register, Information Asset Register, JMFA Staff, RoleIDs, Dependency Map, ROI.

Navigating the graph

  • Scroll to zoom, drag background to pan
  • Click a node to see details; click again or press Escape to deselect
  • Drag nodes to reposition them
  • Use the + / − / ⟳ toolbar buttons to zoom and reset

Default view

On load, only Role and ROI Provider nodes are shown. Tick the Hardware / Software / Information checkboxes in the left panel to overlay those asset types.

Filters

Filters apply immediately and re-run the force simulation. Use the search box to highlight nodes by name or ID.
